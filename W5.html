Netlify en GitHub

Lana van Dessel

Datum: 13 november 2025

In deze les leerden we hoe we onze website online kunnen zetten via Netlify, en hoe we GitHub gebruiken om onze bestanden op te slaan, te updaten en te beheren. Hierdoor kunnen we onze website steeds opnieuw publiceren zonder telkens alles handmatig te hoeven uploaden.

Wat ik heb gedaan

Tijdens de les heb ik een GitHub-repository aangemaakt waarin ik mijn HTML-, CSS- en andere bestanden kan bewaren. Daarna leerde ik hoe ik deze repository kon koppelen aan Netlify, zodat mijn website automatisch wordt geüpdatet wanneer ik nieuwe commits doe.

Ik heb getest hoe pushen en committen werkt. Elke keer wanneer ik iets aanpas in mijn code-editor en dit commit, verschijnt de update automatisch online. Dit maakte het veel overzichtelijker en professioneler om aan websites te werken.

Tot slot heb ik mijn website succesvol online gekregen via een Netlify-link. Hierdoor kan ik mijn werk gemakkelijk delen met anderen.

Reflectie

Ik vond het heel interessant om te leren hoe je een website echt online kunt zetten. Het voelde alsof mijn werk "echter" werd, omdat het niet alleen meer op mijn laptop stond.

In het begin vond ik GitHub best ingewikkeld door de vele termen zoals commits, branches en repositories. Maar na wat oefenen begon het duidelijk te worden en merk ik dat het me veel tijd kan besparen.

Volgende keer wil ik nog leren hoe je meerdere pagina’s beter beheert en hoe je fouten sneller kunt oplossen wanneer Netlify of GitHub niet doet wat ik verwacht.
