Lost in Trainslation - Schetsen

Lana van Dessel

Datum: 18 september 2025

In deze eerste les van Prototyping Tools leerden we wat een prototype precies is en waarom het belangrijk is. We kregen de opdracht “Lost in Trainslation”, waarbij we moesten bedenken hoe we informatie op een treinscherm duidelijk konden weergeven. Het doel was om een overzicht te maken dat voor reizigers makkelijk te begrijpen is.

Mijn eerste schetsen

Ik begon met het maken van schetsen op papier. In het begin dacht ik dat ik alles zelf mocht verzinnen, maar later bleek dat ik een bestaand ontwerp moest verbeteren. Daardoor moest ik even opnieuw beginnen. Nadat ik de opdracht beter had gelezen en uitleg had gevraagd aan klasgenoten, snapte ik beter wat ik moest doen. Toen kon ik gericht werken aan een duidelijker ontwerp.

Tijdens het tekenen merkte ik dat kleine dingen zoals tekstgrootte, kleuren en pictogrammen veel verschil maken. Een goed ontwerp is niet alleen mooi, maar ook overzichtelijk. Dat vond ik een belangrijk inzicht.

Wat ik heb geleerd

Deze les heeft me geleerd dat goed nadenken vóórdat je begint met tekenen veel tijd scheelt. Ik weet nu dat het slim is om eerst te begrijpen wat het doel van de opdracht is, voordat je start met het ontwerp zelf.

Volgende keer wil ik beter letten op de informatie die ik gebruik en eerst een korte planning maken. Ook wil ik oefenen met digitale tools zoals Figma, zodat ik mijn schetsen makkelijker kan uitwerken op de computer.

Aan het einde van de les had ik een duidelijke schets van mijn treinscherm. Het was nog niet perfect, maar ik begreep beter hoe ik een idee kan verbeteren.
