Tailwind en Figma

Lana van Dessel

Datum: 20 november 2025

We hebben eerst één uur uitleg gekregen over hoe we onze website in Tailwind kunnen omzetten. Daarna kregen we websites waarvan we codes konden overnemen om leuke projecten te maken. Vervolgens kregen we één uur een uitgebreide uitleg over hoe we met Figma kunnen werken en kregen we tips over hoe we onze Figma-projecten kunnen verbeteren.

Wat ik heb gedaan

Tijdens de les heb ik eerst geleerd hoe je een bestaande website kunt omzetten naar Tailwind CSS. Ik heb geoefend met het toevoegen van klassen voor kleuren, typografie, marges en layout. Hierdoor kon ik zien hoe Tailwind direct het uiterlijk van de website verandert zonder veel handmatige CSS te schrijven.

Daarna hebben we websites gekregen waarvan we de code konden overnemen. Ik heb deze voorbeelden aangepast om kleine projecten te maken, wat hielp om de logica en structuur van Tailwind-componenten beter te begrijpen.

In het tweede deel van de les heb ik geleerd hoe je met Figma werkt. Ik heb oefening gekregen in het ontwerpen van pagina’s, het aanmaken van componenten, en het toepassen van consistente stijlen. Ook kreeg ik tips om mijn Figma-projecten overzichtelijker en professioneler te maken.

Reflectie

Ik vond het erg nuttig om te leren hoe Tailwind en Figma samenwerken in het webontwikkelingsproces. Tailwind helpt om snel een mooie en consistente website te maken, terwijl Figma ervoor zorgt dat het ontwerp overzichtelijk en professioneel blijft.

Het werken met voorbeeldcode maakte het makkelijker om de theorie direct in praktijk te brengen, en ik merk dat ik nu sneller componenten kan bouwen en aanpassen.

Volgende keer wil ik meer experimenteren met Tailwind en leren hoe ik mijn Figma-ontwerpen nog beter kan vertalen naar een werkende website. Ook wil ik oefenen met het koppelen van design en code zodat alles soepel samenwerkt.
